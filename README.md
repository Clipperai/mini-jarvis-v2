# 🔥 Mini Jarvis v2

A Python-based voice assistant that automates daily tasks like playing music, sending WhatsApp messages, showing notifications, and more.

---

## ⚡ Features

- 🎤 Voice Command Recognition  
- 🔔 Desktop Notifications (plyer)  
- 🎵 Play Music & Videos on YouTube (pywhatkit)  
- 💬 WhatsApp Automation (pywhatkit)  
- 🛠️ Improved Error Handling (stable execution)  

---

## 🧠 Tech Stack

- Python  
- pywhatkit – YouTube + WhatsApp automation  
- plyer – Desktop notifications  
- speech_recognition – Voice input  
- pyttsx3 – Text-to-speech
